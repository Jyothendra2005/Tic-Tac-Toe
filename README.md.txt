# Tic-Tac-Toe Game

A simple Tic-Tac-Toe game built with JavaScript.

## How to Run

Open the `index.html` file in your browser.

## Features

- Two player game
- Simple UI
- Win detection

## How to Play

- Players take turns marking X and O.
- The first to get 3 in a row wins.
